# Predict_Old_Car_Prices
## Overview
A model to predict old car prices using ML models.
Supervised ML models are used here such as linear regression, multiple regression, decision tree, random forest.
 FEATURES USED:
 1. Pandas
 2. Numpy
 3. Matplotlib
 4. Tkinter
 5. Sklearn
